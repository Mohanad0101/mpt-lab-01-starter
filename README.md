# Практика 1: Основы Java

Starter repository for **Практика 1** of the Modern Programming Technologies course.

## Objectives

- Get familiar with basic Java syntax and project structure.
- Implement simple methods and use the `main` entry point for testing.

Course materials: [Modern-Programming-Technologies](https://aliebraheem-fun.github.io/Modern-Programming-Technologies/).

## How to run

From the project root (folder containing `src/`):

```bash
javac src/fa/com/lab1/*.java
java -cp src fa.com.lab1.Main
```

In IntelliJ IDEA: open this folder, mark `src` as Sources Root, then run `Main.java`.

## Tasks

See the TODO comments in `Task1.java` and `Task2.java`. Implement the required logic and run `Main.java` to verify.
